Spade

Application Functionality: Stores and exposes a search API for Highlights comments
The library: provides functionality for querying a comment database and determining the context and location of a comment within an HTML page


Architecture:
- Query Engine
  - Support for regex, normal keyword search and more advanced queries
- Comment Processor
  - Exposes a function to pass in data that contains the selection and the html page on which the selection was made.
    - Generates a unique fingerprint of the comment based on webpage, location on the page, and the text that is highlighted.
